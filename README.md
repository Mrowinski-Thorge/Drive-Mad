# Drive-Mad
The best offline Browser Racing Game

## Features
- Fully functional offline gameplay
- IndexedDB caching for persistent offline access
- Service Worker for reliable offline experience
- Once loaded, always available offline

## Usage
Simply open the website in your browser. On first visit, you'll see a splash screen. After that, the game will be available offline even without an internet connection.
